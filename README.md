### SpringBOOT-React

## Backend Stack:

* Maven;
* Spring Boot, Spring Web;
* Spring Data MongoDB;
* CORS FILTER
* MongoDB embedded database;

#####To run this application use:

```bash
* install and run mongoDB Service
$ cd backend
$ ../mvnw spring-boot:run
```

## Frontend Stack:
* created by create-react-app
* React
* Material UI(downshift), material-ui-icons
* axios

#####To run frontend use:

```bash
yarn install
yarn start
```
